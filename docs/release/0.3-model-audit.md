# Diplomat 0.3.0 Model Audit

This audit covers the curated open-source model registry for Diplomat 0.3.0.

## Registry Scope

The 0.3 model manager exposes only built-in registry entries. Users cannot add arbitrary models from the desktop UI.

## Required Evidence

Before Diplomat 0.3.0 can be accepted:

- Every registry `source_url` must point to a downloadable package artifact or an implemented audited snapshot download path.
- Every registry checksum must be a real SHA256 checksum for the exact downloadable package.
- Every license name and license URL must be present.
- Any attribution obligations must be reflected in release notes or bundled notices.

## Current Gate

The release readiness API intentionally reports model registry blockers while placeholder checksums or bare repository URLs remain. These blockers must be resolved before the 0.3.0 stage gate can be accepted.
