# Diplomat 0.3.0 Packaging Checklist

This checklist defines the Windows desktop packaging path for Diplomat 0.3.0.

## Package Target

- Desktop framework: Tauri 2.
- Product name: Diplomat.
- Version: 0.3.0.
- Windows installer target: NSIS.
- Bundle icons:
  - `apps/desktop/src-tauri/icons/icon.ico`
  - `apps/desktop/src-tauri/icons/icon.png`

The release asset verifier enforces this metadata:

```powershell
node .\scripts\verify-release-assets.mjs
```

## Build Steps

Run from the workspace root:

```powershell
$env:DIPLOMAT_FFMPEG_PATH = (Resolve-Path ".\.dev\tools\ffmpeg-release-essentials\ffmpeg-8.1.1-essentials_build\bin\ffmpeg.exe").Path
$env:DIPLOMAT_FFPROBE_PATH = (Resolve-Path ".\.dev\tools\ffmpeg-release-essentials\ffmpeg-8.1.1-essentials_build\bin\ffprobe.exe").Path
corepack pnpm install --frozen-lockfile
corepack pnpm --dir apps/web build
corepack pnpm --dir apps/desktop tauri build
```

If the Tauri CLI is unavailable through the package manager, install the matching Tauri v2 CLI before building. Do not change the bundle target to a non-Windows target for the 0.3 release.
The release runtime must either set these environment variables or bundle equivalent release-approved FFmpeg and FFprobe binaries.

## Installer Review

After the build finishes:

- Confirm the generated installer name includes Diplomat and 0.3.0.
- Install into a clean Windows user profile or VM.
- Launch the app from the Start menu.
- Confirm the app creates runtime directories under `%LOCALAPPDATA%\Diplomat`.
- Confirm Settings can open data, model, and log directories in File Explorer.
- Confirm Worker startup either succeeds or reports an actionable port/Python error.
- Confirm Settings and `GET /release/readiness` report the release FFmpeg and FFprobe paths as available.

## Artifact Handling

- Keep the installer, release readiness JSON, and smoke-test notes together.
- Do not publish unsigned builds as final unless the release note explicitly marks them as unsigned.
- Do not ship a build whose readiness report contains blockers.
